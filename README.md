## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

## Before you start

- [ ] Install [Node.js](https://nodejs.org/en/) (v14.17.0 or higher)
- [ ] Run sql script in `./sql` folder
- [ ] Modify .env file in root folder with own data to access database
